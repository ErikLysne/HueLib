    Detailed Description

    Contains the details of reply received from HueBridge::sendRequest().

    HueReply holds the JSON, HTTP status, and error returned from HueBridge::sendRequest(). Also holds booleans specifying if the reply is valid and if it timed out.

    In case a GET request was sent, JSON contains the raw JSON reply. If a PUT or POST request was sent successfully, or GET/ PUT/ POST request returned with a JSON error, JSON contains the object contained within the success/ error reply (i.e. JSON is stripped of the outer success / error brackets). This is becasue success and error JSON are returned as JSON arrays while replies to GET requests are returned as JSON objects. JSON is always a QJsonObject.

    HTTP status is the HTTP status of the reply. If the status is not 200, the reply is not valid.

    Note: replies with 200 status can still contain JSON errors. Instead of testing for 200, use isValid(). HTTP status is an int.

    error contans details about any errors in the reply, and is a HueError object.

    Overloads QString() casting.
