    Detailed Description

    Contains the details of request sent to HueBridge::sendRequest().

    HueRequest holds the URL path extention, JSON and method sent to HueBridge::sendRequest().

    The URL path extention will be appended to the base URL which contains the IP address and username of the bridge. It determines which resource to address and which property to retrieve/modify. Is a QString.

    JSON is the body of the request. It specifies modifications on the bridge.

    Method is either GET, PUT or POST.
