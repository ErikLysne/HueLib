    Detailed Description

    HueAbstractObject is the abstract base class for HueLight and HueGroup.

    HueAbstractObject specifies the common interface for HueLight and HueGroup. The class implements a series of public set functions that changes properties of the derived class objects by sending requests to HueBridge. The set functions call corresponding update functions which are pure virtual and must be overloaded by the derived classes.
