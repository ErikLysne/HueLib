    Detailed Description

    Contains the details of an error received from HueBridge::sendRequest.

    HueError contains the type, address and description of an error returned from HueBridge::sendRequest. type is an ID of the error and is an int. The easiest way to test for specific errors is by comparison with type. address contains the path extention to the resouce that produced the error and is a QString. description provides a readable description of what caused the error and is a QString.

    A list of errors can be found in https://developers.meethue.com/develop/hue-api/error-messages/.
