• HueLib

    • HueLib

    Contains classes for interfacing with Philips Hue lights.

    Intended as a high-level library where little knowledge about the Philips Hue API is required.

    HueAbstractObject

    The abstract base class for HueLight and HueGroup

    HueBridge

    Represents the physical bridge connected to the network

    HueError

    Contains the details of an error received from HueBridge::sendRequest()

    HueGroup

    Used to interface with a group of Hue lights

    HueLight

    Used to interface with a single Hue light

    HueReply

    Contains the details of reply received from HueBridge::sendRequest()

    HueRequest

    Contains the details of request sent to HueBridge::sendRequest()

    Group

    Contains data classes for HueGroup

    Light

    Contains data classes for HueLight

    Group::Action

    Describes the action of a HueGroup

    Light::Config

    Describes the config of a HueLight

    Group::GroupClass

    Describes the class of a HueGroup

    Group::Lights

    Describes the lights in a HueGroup

    Light::Manufacturer

    Describes the manufacturer of a HueLight

    Group::Name

    Describes the name of a HueGroup

    Light::Name

    Describes the name of a HueLight

    Light::ProductID

    Describes the product ID of a HueLight

    Light::ProductName

    Describes the product name of a HueLight

    Group::Recycle

    Describes the recycle of a HueGroup

    Group::Sensors

    Describes the sensors in a HueGroup

    Light::SoftwareConfigID

    Describes the software config ID of a HueLight

    Light::SoftwareUpdate

    Describes the software update of a HueLight

    Light::SoftwareVersion

    Describes the software version of a HueLight

    Light::Config::Startup

    Describes the startup in Config of a HueLight

    Group::State

    Describes the state of a HueGroup

    Light::State

    Describes the state of a HueLight

    Group::Type

    Describes the type of a HueGroup

    Light::Type

    Describes the type of a HueLight

    Light::UniqueID

    Describes the unique ID of a HueLight
