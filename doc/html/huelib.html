• HueLib

    • HueLib

    Contains classes for interfacing with Philips Hue lights.

    Intended as a high-level library where little knowledge about the Philips Hue API is required.

    HueAbstractObject

    The abstract base class for HueLight and HueGroup

    HueBridge

    Represents the physical bridge connected to the network

    HueError

    Contains the details of an error received from HueBridge::sendRequest()

    HueGroup

    HueLight class is used to interface with a group of Hue lights

    HueLight

    Used to interface with a single Hue light

    HueReply

    Contains the details of reply received from HueBridge::sendRequest()

    HueRequest

    Contains the details of request sent to HueBridge::sendRequest()
